# ランク制度

## 等級とレベルの対応

人も妖怪も同じ基準で等級が定められる。等級は、ファンタジーRPG的な戦闘力の基準となる「レベル」に対応する。

| 等級 | レベル |
|---|---|
| F級 | 1〜5 |
| E級 | 6〜20 |
| D級 | 21〜40 |
| C級 | 41〜60 |
| B級 | 61〜80 |
| A級 | 81〜 |

等級・レベルの目安は人にも妖怪にも共通して適用され、基礎スペック差として扱う（能力の再現度との関係は「能力体系.md」を参照）。

## 人口規模の目安

等級人口は、一階層上がるごとに約10分の1に減っていく。日本国内では、人類側・妖怪側とも、おおよそ次の人数を目安とする。

| 等級 | 人数の目安 |
|---|---|
| F級 | 200万人 |
| E級 | 20万人 |
| D級 | 2万人 |
| C級 | 2000人 |
| B級 | 200人 |
| A級 | 18人 |

F級適性者は、一般人口のおよそ50〜100人に1人程度の割合で存在するとされる。

## 国際的な呼称

等級には国際共通の呼び方がある。この国際等級制度は約100年前に海外主導で制定されたもので、各国・各勢力・各文化圏によって実際の運用や呼び方には差がある。日本国内では、この国際呼称と国内呼称（F級〜A級）が並行して使われている。

- F級＝視認者：超常存在を知覚できる最低ラインの適性者。正式な戦闘職ではないが、違和感の感知・監視・呪具の保守・地域協力など、現代超常社会の最前線の観測層を支える。一般人には気づけない違和感や小規模霊障も、F級適性者には「何かがおかしい」と分かることが多い。

  F級が重要なのは、専用の超常機関にだけいるからではない。50〜100人に1人という頻度は、役所・警察・学校・病院・企業といったごく普通の組織の窓口や現場に、ほぼ必ず誰か一人はF級適性者が紛れ込んでいる密度である。だからこそ、霊務省や退魔学校のような専門組織を経由しなくても、社会のあらゆる末端で「何かおかしい」が最初に拾われ、必要な専門家へ繋がる。妖怪が実在する社会が、専用インフラを組まなくても現代社会の形のまま回っていられるのは、この分布密度が前提になっている。F級がこの頻度でいなければ、妖怪のいる社会は今の形では成立しない。

  F級として認定されると、専用の手帳が交付され、霊務省の連絡網に登録される。何か異変を見つけた際はこの連絡網を通じて通報・共有される。F級認定は履歴書にも記載できる公的な資格であり、就職や職場での扱いにおいて、あくまで「それなり」の範囲ではあるが優遇される（採用時の一定の評価、職場での担当割り当てなど）。特別扱いというほどではないが、社会的に意味のある資格として機能している。

  F級適性は、専用の検査を受ければ簡単に判明する程度のものであり、深く秘匿された才能ではない。乳幼児健診のタイミングで調べておく家庭もあれば、「うちの子、妖怪が見えているみたいだ」と保護者が役所へ相談を持ち込み、そこから検査に繋がるケースも多い。いずれにせよ発見自体は難しくなく、検査を受けるかどうか・いつ受けるかという各家庭の判断の差が、認定時期のばらつきを生んでいる。
- E級＝初歩術者：初歩的な術式・呪具運用ができる準戦力。ここから正式に術を使えるようになる。学校・地域・神社・企業の現場実務を広く支える。

  「初歩」という呼び方は戦闘力としての等級を指すのであって、技能の未熟さを意味しない。E級には、戦闘にこそ向かないが治療・結界構築・土地の浄化・呪具作成など特定分野を何十年も突き詰めた大ベテランの術師も多く含まれる。A級が国家規模の抑止力を担う大企業の社長だとすれば、この手のE級は、その道何十年の町工場の職人のようなものである。等級は総合的な戦闘力の目安であり、専門分野の熟練度そのものを表す指標ではない。
- D級＝正式戦力：一般的な怪異へ単独・少人数で対応できる正式な実務者。通常の案件処理網は主にD級によって維持されている。
- C級＝精鋭：特殊能力を持つ妖怪・条件怪異へ実戦対応できる水準。崩れた現場を立て直す層として、専門分野や国家案件へ重点的に投入される。
- B級＝国家級：都市機能や広域案件に影響し得る戦力。単独で地域の戦力バランスに影響し得るため、行政・旧家・民間・海外機関から勧誘・監視の対象になりやすい。
- A級＝戦略級：国家の抑止力に相当する戦略戦力。希少だが、伝承・歴史・組織の蓄積がある限り実在する。

## 単純な破壊力の目安

条件・領域・契約・認識干渉を除いた、単技一発あたりの破壊力の目安。

| 等級 | 破壊力の目安 |
|---|---|
| E級 | 鍛えた人間並みの身体能力を持つ低級妖怪程度 |
| D級 | 大型猛獣に属性攻撃や硬い外殻・突進などが乗った程度 |
| C級 | 車・コンクリート壁・建物構造の一部を壊せる程度 |
| B級 | 一つの技でビルを破壊できる程度 |
| A級 | 一つの技で山を破壊するか、山岳規模の地形を変えてしまう程度 |

ただしこれはあくまで平均的な火力換算の目安であり、契約・領域維持・認識干渉など火力以外へ能力を振っている場合はこの限りではない。同じA級でも、山を砕く者、死を告げる者、契約を強制する者では危険の方向性がまったく異なる。

## 高位戦力の内訳

A級18人・B級200人も、退魔師戦力全体と同様に行政・神社・民間でおおむね3等分される（詳細は「組織.md」を参照）。民間側には合法な事業者だけでなく、グレー・非合法な裏社会勢力も含まれる。国内でA級相当の重犯罪者としてマークされているのは六人前後とされ、これは民間枠のうち裏社会に区分される者に対応する。

## 担当範囲

- F級・E級：学校職員、民間業者、地域の監視員などが関わりやすい。
- D級：正式な実務者、民間の超常企業、神社などが合同で対応する。
- C級：専門部隊や上位者への引き継ぎが視野に入る。学生は補助・記録にとどめる。
- B級以上：行政の広域対応、複数組織の調整が絡む。
- A級以上：国家規模の判断が必要になる。

初手で相手の危険度が正しく認識されなかった場合、行政上の対応はD級退魔師→C級退魔師→B級退魔師という具合に、原則として一段ずつ階級を上げてお鉢が回る。等級を飛ばして一足飛びにB級やA級を投入することはせず、緊急性が高い場合はこの段階を飛ばすのではなく、各段階の判断・引き継ぎの速度を上げることで対応する。

ただし、上位への引き継ぎは単に「被害が出た」というだけでは発生しない。死者数がせいぜい数人程度に収まるような被害では、C級への案件送りにはならない。C級以上へ引き継がれるのは、数十人規模・施設や地域単位に及ぶ被害、被害の反復・拡大・長期化、下位の実務者が調査・制圧に失敗し撤退や連絡途絶に至った事実、あるいは条件戦・認識干渉といったC級以上の専門性そのものが必要な性質を持つ場合である。少人数の被害と「不気味だ」「なんとなく嫌な予感がする」だけでは、上位への案件送りの根拠にはならない。

## 誤判定と撤退

現場では等級の誤判定が起こり得る。低い等級だと思って対応した相手が、実際にはより上の等級だったという事態は珍しくない。F級だと思ったらE級の群体だった、E級だと思ったらD級の縄張り個体だった、D級だと思ったらC級の入口だった、というような段階的な見誤りは典型例である。反応量が低く見えても、媒体分離や契約条件によって危険度が上振れする場合もある。

そうした場合は無理に制圧を続けず、退避・封鎖・記録・上位への引き継ぎを優先する。撤退は敗北ではなく、被害を広げず専門家へ引き継ぐための正しい判断として扱われる。

## 特殊A級

レベル100を超えた妖怪にのみ表記される分類。災害そのものとほぼイコールの存在であり、人間はレベル100を超えても特殊A級にはならない。

「人口規模の目安」の表はあくまで日本国内の数字であり、A級18人からさらに10分の1にした1〜2体程度というのは、日本国内で確認される特殊A級の希少さの根拠である。日本国内で正式に確認されているのは、玉藻前（封印中）、酒呑童子（消息不明）の二体のみで、このピラミッド構造にちょうど一致する。

過去に《ベルゼブブ戦役》と呼ばれる事件が起きて以降、特殊A級妖怪は国際超常社会における最重要監視対象として扱われている。

この10分の1ピラミッドを世界規模へ拡張すると、特殊A級に相当するのは各国・各宗教圏に伝わる世界的伝承・文明象徴として極めて有名な存在であり、日本の2体はその一国分の内訳にすぎない。日本一国だけでA級18人・特殊A級2体が存在する以上、世界全体のA級・特殊A級の総数は各国・各文明圏の分だけその何倍にもなる。

## 未分類超高位存在群

特殊A級のさらに上には「未分類超高位存在群」という仮説的な階層が想定されている。これは世界規模に拡張した特殊A級の総数から、さらに10分の1にした領域である。特殊A級自体が世界規模では相応の数（各国・各文明圏の分）に上る以上、未分類超高位存在群も「0〜1体」のような極端な希少さではなく、両手で数えられる程度の複数体が世界に存在すると見込まれる。ヤハウェ・仏陀・ガイア・地獄などの名が仮説例として複数並んで挙げられるのは、この規模感とおおむね整合する。

ただし、これは単なる強さのランキングの延長ではなく、文明・信仰・死生観・土地概念・思想体系そのものへ近い、一国では収まらない世界規模の確定不能な領域として扱われる。個々の実在性・能力・序列は確定しない（詳細は「禁則.md」のUNKNOWN領域を参照）。

## 等級外・測定不能

特殊A級とは別に、既存の等級へ正常に収まらない存在に付与される管理区分として「等級外・測定不能」がある。

これは単純に「A級より強い」という意味ではない。観測不能・測定不能・情報不足・存在不安定・一時的な超常暴走など、既存の分類そのものが正常に適用できない場合を含む管理上の区分であり、性質や危険度は個体ごとに大きく異なる。UNKNOWN近傍の存在については、実在性・能力・序列・規模を確定しない。

## 等級で見る対象

等級は妖怪・人間・呪具のいずれにも使われるが、対象によって見ている中身が異なる。

- 妖怪の等級：戦闘能力・危険度・能力・存在構造を中心に見る。
- 人間（退魔師）の等級：本人の出力だけでなく、装備運用・知識・現場対応力を中心に見る。
- 呪具の等級：出力・危険性・使用条件・管理難度を中心に見る。

同じ「C級」でも、何の等級の話をしているかによって意味が変わる点に注意する。

## 運用上の注意

等級・レベルは基礎スペック差の目安であり、実際の勝敗を保証するものではない。妖怪の能力は伝承・役割の範囲を超えないため、条件や相性次第では等級が低い側が優位に立つ場合もある。
