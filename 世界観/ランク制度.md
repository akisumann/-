# ランク制度

## 等級の性質

F級〜A級の等級は、危険度・対処能力・社会的重要度・管理コストを含む目安である。

等級と戦闘力は無関係ではなく、多くの場合、基礎スペック・出力・耐久・速度・攻撃能力・防御能力などの戦闘力も概ね等級に準ずる。ただし妖怪の能力・条件・相性は基礎スペック差を覆す場合があるため、等級は勝敗保証ではない。

人類側の等級は、個人出力だけでなく術式・装備・知識・準備・封印技術・組織運用を含めた総合的な対処能力を示す。

## 等級区分

| 等級 | 目安 |
|---|---|
| F級 | 固有能力を持たない最下位の怪異。数・習性・環境適応で危険性を持つ |
| E級 | 単純な固有能力を持ち始める段階。地域実務者が単独で対処可能 |
| D級 | 一般的な退魔師・術師が標準装備で対応する主力帯 |
| C級 | 複数人での連携や、専用呪具・霊装の運用が前提になる帯 |
| B級 | 地域を超えた被害が想定され、行政・広域実務者の関与が視野に入る |
| A級 | 都市規模の被害・国家的関心事となる案件。専門部隊での対応が原則 |
| 特殊A級 | A級の中でも突出した個体・現象。国家級の管理体制と情報統制を伴う |
| 等級外 | 既存の等級体系・分類・対応体系で説明しきれないUNKNOWN領域の存在 |

## 運用上の注意

等級は現場での勝敗を保証するものではない。突発遭遇・情報不足・装備不足・相性不利・条件不明があれば、同格以下の相手にも敗北し得る。等級外の存在は軽率に確定説明せず、UNKNOWN領域として扱う（詳細は「世界設定.md」を参照）。
