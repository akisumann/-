# 男性登録制度

## 登録制度

希少な男性は、成人に達すると公的機関への登録が義務付けられる。登録により、健康状態・遺伝的な傾向・精力水準などが検査され、公的な記録として管理される。この登録は男性を保護すると同時に、社会的な子作り需要と結びつける仕組みでもある。

## 種付け報酬

登録された男性が女性との間にそういった行為（いわゆる「種付け」）を行った場合、男女どちらにも公的機関から一定額の報酬が支払われる。子が生まれるかどうかに関わらず行為そのものに対して支払われる仕組みであり、積極的に行為へ臨む者ほど受け取る金額が多くなる。これは対価性のある正式な制度として運用されており、恥ずべきものではなく、男女双方にとって名誉と収入を兼ねた社会的な役割として広く認知されている。

## ランク制度

登録された男性には、検査結果に基づいてE〜Sの6段階のランクが付けられる。人数はE級・D級が大半を占め、上位ランクに上がるほど急激に希少になるピラミッド構造になっている。

ランクを分ける最大の指標は、精力の持続・回復力である。

| ランク | 目安 |
|---|---|
| S級 | 1日に何度でも可能 |
| A級 | 1日1回可能 |
| B級 | 週1回可能 |
| C級 | 月1回可能 |
| D級 | 射精は可能と言えば可能、という水準 |
| E級 | 不可能 |

ランクが高いほど種付け報酬も高くなり、また社会的な需要（指名・優先枠など）も集中しやすい。ウイルス禍以前の水準の精力を持つ人間は、この世界の基準ではS級の中でもさらに突出した水準に相当する。

## 人口規模の目安

総人口は約6,000万人（戦争と長期的な人口減少により、現実の日本のおよそ半分程度に減少している）。男性比率は数百人に1人（およそ1:300）で推移しており、男性総数はおよそ20万人。

| ランク | 割合 | 人数目安 |
|---|---|---|
| S級 | 0.1% | 200人 |
| A級 | 0.9% | 1,800人 |
| B級 | 4% | 8,000人 |
| C級 | 15% | 3万人 |
| D級 | 30% | 6万人 |
| E級 | 50% | 10万人 |

人口維持には年間およそ60万人の出生が必要となる。C級以上（実質的に子作りへ貢献できる層、合計約4万人）が平均して年間15人前後の女性を妊娠させることで、この必要出生数がちょうど成立する計算になっている。S級・A級のような高頻度ランクほど担当する女性の数が多く偏っており、この平均値を支えている。

## ランク検査

ランク検査は特別な儀式めいたものではなく、通常の健康診断の一項目として扱われる。学校の身体測定、就職時の健康診断、かかりつけ医での定期検診など、様々なタイミングで自然に組み込まれており、専用の検査施設にわざわざ出向く必要はない。

結果を告げる医師や検査員の態度も、血圧や視力の結果を伝えるのと変わらない。「血圧、ちょっと高めですねー」というのと同じ温度感で「ランクはC級ですねー」と告げられる。深刻に扱われることも、過度に持ち上げられることもない、ごくありふれた検査項目の一つとして社会に定着している。

ランクは一度きりの判定ではなく、加齢・体調・生活習慣などによって変動しうるため、定期検診のたびに更新される。
